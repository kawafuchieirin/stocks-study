# stocks-study

## 概要

本リポジトリは、株式市場のデータを取得し、分析・検証を行うことで株式投資の理解を深めることを目的とした学習用プロジェクトです。

株価データの収集、テクニカル指標の算出、戦略の検証（バックテスト）などを段階的に実施します。

---

## 目的

* 株式市場の仕組みを理解する
* 株価データの取得方法を学ぶ
* データ分析スキルを向上させる
* 投資戦略を検証する
* 分析結果を記録し、改善を行う

---

## 主な学習内容

### データ取得

* 株価データの取得
* 財務データの取得
* マーケット指標の取得

### データ分析

* 時系列分析
* テクニカル分析
* 財務指標分析

### 戦略検証

* 売買ルールの設計
* バックテスト
* パフォーマンス評価

---

## 使用技術

### 言語

### 主なライブラリ

* pandas

## 分析フロー

1. 株価データを取得する
2. データを整形する

4. 売買戦略を作成する
